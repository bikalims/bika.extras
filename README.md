# Bika Extras

**Additional useful features and extensions for Bika LIMS**

This package provides minor extra functionality, tools, and enhancements that are not part of the core Bika LIMS but are commonly used by laboratories.

### Installation

Add `bika.extras` to your Bika LIMS or SENAITE instance and restart.
